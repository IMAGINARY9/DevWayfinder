> **Version:** 1.0.0  
> **Status:** Active  
> **Last Updated:** 2026-04-01  
> **Authoritative Source:** This document is the single source of truth for benchmark methodology and published baseline results.

---

## 1. Benchmark Scope

MVP 3 benchmark suite measures:

- Analysis throughput (project scan + dependency graph)
- Full guide generation time in heuristic mode
- Peak Python memory usage (`tracemalloc`)

Benchmarks run on synthetic fixtures generated by the repository runner:

- `small` target: 10 modules
- `medium` target: 100 modules
- `large` target: 1000 modules

Observed module counts are slightly higher because each fixture also includes package `__init__.py` files and `main.py`.

## 2. Reproduction

From repository root:

```bash
# PowerShell example
$env:PYTHONPATH = "src"
.\.venv\Scripts\python.exe benchmarks/run_benchmarks.py --force-regenerate
```

Generated artifacts:

- `benchmarks/results/latest.json`
- `benchmarks/results/latest.md`
- `benchmarks/fixtures/generated/{small,medium,large}`

## 3. Published Baseline (2026-04-01)

| Size | Modules Detected | Dependencies | Analysis (s) | Generation (s) | Total (s) | Peak Memory (MB) |
|------|------------------|--------------|--------------|----------------|-----------|------------------|
| small | 13 | 13 | 0.054 | 0.035 | 0.089 | 0.20 |
| medium | 109 | 116 | 0.178 | 0.238 | 0.416 | 1.08 |
| large | 1021 | 1144 | 1.457 | 1.929 | 3.387 | 9.82 |

Source artifact: `benchmarks/results/latest.json`

---

## 4. Interpretation

- NFR target for 1000-file analysis (`< 3 minutes`) is met by a large margin in heuristic mode.
- Memory growth is near-linear and remains well below the 500 MB requirement threshold.
- End-to-end guide generation for large synthetic fixtures remains under 4 seconds without external LLM calls.

---

## 5. Caveats

- Results are heuristic-mode baselines (`--no-llm` equivalent); live-provider runs vary by model latency and network conditions.
- Synthetic fixtures reflect dependency-chain and multi-package patterns, but do not model every real-world framework shape.
